About the Project

SmartBot is a state-of-the-art AI-powered robot that integrates Internet of Things (IoT) technologies and robotics. With voice interaction, surveillance capabilities, and remote control, SmartBot offers an intuitive and advanced solution for various applications.

Features

AI-Driven Decisions

SmartBot utilizes AI algorithms for real-time decision-making and autonomous actions based on the environment.

Voice Interaction

Interact with SmartBot using voice commands, and hear responses via speech synthesis, creating a more immersive experience.

Surveillance with ESP32-CAM

Live video streaming for security and monitoring, accessible remotely via Wi-Fi or Bluetooth.

Applications

Smart Homes

Enhance your home with automation, surveillance, and voice-assisted controls.

Security Systems

Utilize SmartBot for real-time video streaming and remote surveillance in various environments.

Industrial Automation

SmartBot assists in automating repetitive tasks, monitoring industrial environments, and improving safety.
