About the Project

SmartBot is a state-of-the-art AI-powered robot that integrates Internet of Things (IoT) technologies and robotics. With voice interaction, surveillance capabilities, and remote control, SmartBot offers an intuitive and advanced solution for various applications. This robot is capable of making decisions in real-time through AI algorithms, interacting with users via voice synthesis, and streaming live video for remote monitoring. The combination of these features provides an immersive and highly functional experience, making SmartBot ideal for smart homes, security systems, and industrial automation.

Features

AI-Driven Decisions

SmartBot utilizes AI algorithms for real-time decision-making and autonomous actions based on the environment.

Voice Interaction

Interact with SmartBot using voice commands, and hear responses via speech synthesis, creating a more immersive experience.

Surveillance with ESP32-CAM

Live video streaming for security and monitoring, accessible remotely via Wi-Fi or Bluetooth.

Smart Homes
